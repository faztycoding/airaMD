# Diagrams

This folder contains clinical face diagram overlays used in the Face Diagram feature.

## Required Assets (replace with production versions)

- `face_front.svg` — Front-facing face outline
- `face_left.svg` — Left profile face outline
- `face_right.svg` — Right profile face outline
- `face_lip.svg` — Lip zone detail outline

> **Note:** The face diagram images in `assets/images/face_diagrams/` are the actual
> raster images used as drawing canvases. This folder is for SVG overlays/annotations.
